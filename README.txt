	Блог. Версия 1.0

Проект представляет собой веб-приложение на базе фреймворка Laravel, предназначенное для работы с сессиями и куки.
Это приложение позволяет пользователям создавать, просматривать и удалять данные о сессиях и куки.


-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------


	Установка и настройка

1. Клонируйте репозиторий проекта на свой локальный компьютер:

    git clone <ссылка на репозиторий>

2. Перейдите в директорию проекта:

    cd <название_директории>

3. Установите зависимости с помощью Composer:

    composer install

4. Выполните миграции для создания таблиц в базе данных:

    php artisan migrate

	Примеры использования

Создание новых куки/сессий.

1. Перейдите на главную страницу.
2. Введите ключ.
2. Нажмите кнопку "Сохранить данные сессии/Сохранить данные куки".

Просмотр информации о куки/сессии.

1. Перейдите на главную страницу.
2. Введите ключ.
2. Нажмите кнопку "Получить данные сессии/Получить данные куки".

Удаление информации о куки/сессии.

1. Перейдите на главную страницу.
2. Введите ключ.
2. Нажмите кнопку "Удалить данные сессии/Удалить данные куки".
